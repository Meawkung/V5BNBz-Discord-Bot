# üî∂ **Discord Bidding Bot User Guide (Updated)**

Welcome to the Bidding Bot! Here‚Äôs how to use it:

### **How to Bid on an Item**

1.  **Select an Item:** See an item you want in the bidding message? **Click the button with the item's name** (e.g., the `Netherforce` button).
2.  **Increase Your Bid:** **Each time you click the button, it adds one bid entry for you.** Your name will appear in the list for that item.
3.  **Check Status:** The bot automatically updates the main message after every action, so everyone can see the latest bid list.

### **Manage Your Bids**

You have full control over your own bids using the buttons at the bottom:

-   **`Clear My Bids`:**
    When clicked, the bot will ask for your choice:
    *   **Clear ALL My Bids:** Removes **every single bid** you have placed across all items.
    *   **Clear ONLY Done Bids:** Removes only the bids that you have already marked as complete (those with a ‚úÖ).

-   **`Done Bidding`:**
    Use this when you have successfully received an item from a bid. It's a simple 2-step process:
    1.  **Step 1:** Use the first dropdown menu to **select the *item*** you want to manage.
    2.  **Step 2:** A second dropdown will appear. Use it to **select the *specific bid(s)*** you have completed. You can select multiple bids at once. The bot will then mark those entries with a ‚úÖ.

-   **`üîÉ` (Refresh):**
    If the bid list seems out of date, click this button to manually refresh it with the latest data.

### **Admin Controls**

Admins have extra tools to manage the bidding system:

-   **`Restart Bidding` (Button):** Admins can click this button to **clear all current bids for everyone** and completely reset the system.
-   **`!pause` (Command):** Type this command to **temporarily pause all bidding.** Users won't be able to click the bidding buttons.
-   **`!resume` (Command):** Type this command to **resume bidding** after it has been paused.
-   **`!manualbid` (Command):** Manually sets or overrides the total number of bids for a specific user on an item.
    *   **Example:** `!manualbid "Netherforce" @SomeUser 3`
